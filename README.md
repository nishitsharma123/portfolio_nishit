## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/nishitsharma123/portfolio_nishit.git
```

2. Navigate to the project folder:

```bash
cd portfolio_nishit
```

3. Open `index.html` in your browser.
